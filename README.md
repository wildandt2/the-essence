# ☕ The Essence POS System

A modern **Point of Sale (POS) web application** for cafe businesses built with **Next.js, Express.js, and Docker**.

---

## 🚀 Features

- 🔐 Authentication (Login with JWT)
- ☕ Menu display (Coffee, Snack, etc.)
- 🛒 Cart system (Add items to order)
- 💰 Checkout system
- 📊 Dashboard UI (modern cafe design)
- 📱 Responsive (mobile-friendly)
- 🐳 Docker support

---

## 🖼️ Preview

> UI inspired by modern cafe POS systems

- Clean white + cafe aesthetic design
- Product cards with images
- Order panel (cart) on the right
- Simple and intuitive layout

## 🧱 Tech Stack

### Frontend

- TypeScript

### Backend
- Node.js

- Prisma ORM

### Database
- PostgreSQL (via Docker)

- Docker & Docker Compose
- Git & GitHub

## ⚙️ Installation & Setup

### 1. Clone Repository

```bash
git clone https://github.com/USERNAME/the-essence-pos.git
cd fullstack-app
2. Run Backend
cd backend
npm install
nodemon index.js

Backend running at:
http://localhost:5000

3. Run Frontend
cd frontend
npm install
npm run dev

Frontend running at:
http://localhost:3000

🐳 Run with Docker (Optional)
docker-compose up --build

💰 Currency
All prices are displayed in Indonesian Rupiah (IDR).
